## Run Locally

```bash
# Clone the repo
git clone https://github.com/Yashwanth-K/portfolio.git

# Navigate to project
cd portfolio

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

---

## Deployment

Deployed on [Vercel](https://vercel.com). Every push to `main` triggers automatic deployment.
